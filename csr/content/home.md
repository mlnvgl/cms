# Welcome to the CMS

This is a simple client-side rendered CMS powered by **React**, **Vite**, and **Markdown**.

All content lives in `.md` files and is rendered at build time into your app.

## Features

- Write content in Markdown
- Frontmatter metadata (title, slug, etc.)
- Client-side routing
- Fast Vite dev server
