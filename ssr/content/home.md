# Welcome to the SSR CMS

This is a server-side rendered CMS powered by **Express**, **EJS**, and **Markdown**.

Content lives in `.md` files and is rendered to HTML on each request by the server.

## Features

- Write content in Markdown
- Frontmatter metadata (title, slug, etc.)
- Server-side rendering with EJS templates
- Full HTML delivered to the browser -- no JavaScript required
